Features

  • Create zipped backups of your server files
  • Download created backups to your computer
  • Rollback a particular backup (restore points)
  • Create database backup
  • Delete backup files
  • Download created backup  (automatically zipped for faster download)
  • Rollback a particular database backup (restore points)
  • Database SQL importer
  • Exclude directories, files, file extensions from your backup to make the backups smaller
  • Cronjob supported to create file backups or restore them
  • Cronjob supported to create database backups or restore them
  • Easy to install (change config and upload to server)
  • Easily creating backup from your, and your clients websites.
  • Email file and/or database backups to your email address
  • Backup to remote FTP server
